# Bakery API Project
## Overview
This is a sample project using Cloudflare Workers, Wrangler, D1 database, and Workers AI. It was developed to demonstrate the various tools that are built into Cloudflare for developers to create on the Cloudflare platform.  

## Tools
Cloudflare workers: Used for hosting serverless compute.  It can directly deploy a project from Github fully onto the Cloudflare network.  Includes automatic builds, versioning, and custom domains.
Wrangler: Cloudflare's CLI tooling for developing and deploying onto Cloudflare.
D1 database: SQL-lite hosted database. Great for small relational databases.
Workers AI: An gateway to Cloudflare's hosted AI models. Build AI tools directly into your projects on the Cloudflare network.
